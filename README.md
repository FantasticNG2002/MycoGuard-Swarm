# 🍄 MycoGuard AI: Secure Multi-Agent Orchestration

## 📖 Project Overview
In biological data science, a single "False Negative" in mushroom classification (misclassifying a poisonous specimen as edible) can be fatal. Standard AI coding assistants fail to enforce rigorous, domain-specific safety checks. 

**MycoGuard** solves this by deploying a collaborative swarm of expert AI agents powered by **Google ADK** and **Gemini 2.5-flash**. It automates the generation, training, and strict safety auditing of machine learning pipelines.

---

#🛠️ Prerequisites 
Before running this project, you need to prepare the following:

Google Gemini API Key: Get it from Google AI Studio.

Ngrok Auth Token: Sign up at Ngrok to get a free auth token (required to expose the Colab local server to the public web).

Dataset: Ensure mushroom(1).csv is uploaded to your Colab workspace or local directory.

## 🚀 Quick Start (Recommended: Colab)
The easiest way to review and run this project is via the provided Jupyter Notebook.

open MycoGuard_Live_Demo.ipynb

How to configure the Notebook:
Open the .ipynb file in Google Colab.

Run the environment setup cells (!pip install fastapi uvicorn pyngrok google-adk google-generativeai).

API Key Setup: When you run the API Key cell, a secure prompt will appear. Paste your Gemini API Key there.

Ngrok Setup: In the FastAPI Server cell, locate the following line and replace the placeholder with your actual Ngrok token:

Python
NGROK_AUTH_TOKEN = "YOUR_NGROK_AUTH_TOKEN_HERE"
ngrok.set_auth_token(NGROK_AUTH_TOKEN)
Run the server cell. It will output a public URL (e.g., https://xxxx.ngrok-free.app/generate).

## 💻 Running the Frontend UI
We have provided a lightweight HTML frontend (/frontend) to interact with the agent swarm.

## ⚠️ Crucial Step: Connecting Frontend to Backend
Before opening the HTML file, you MUST update it with your active Ngrok URL:

Open frontend/index.html in any text editor.

Locate the JavaScript section at the bottom.

Replace the NGROK_URL variable with the URL generated by your Colab backend:

JavaScript
// Change this to your active Ngrok URL
const NGROK_URL = "https://YOUR_NEW_NGROK_ID.ngrok-free.app/generate"; 
Save the file and simply double-click index.html to open it in your browser. Enter your prompt and watch the agents collaborate!

Alternative Installation:
Run pip install -r requirements.txt to install all necessary dependencies for the MycoGuard AI swarm.

## 📂 Repository Structure
/agents/: Contains the modular code defining the Router, Auditor, Forge, and Validator agents (Google ADK configurations).

/app/: Contains main.py, the FastAPI backend gateway managing incoming requests and triggering the agent workflows.

/frontend/: Contains the lightweight index.html UI for user interaction.

MycoGuard_Live_Demo.ipynb: The complete, runnable Google Colab notebook demonstrating the entire pipeline.
